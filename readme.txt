This is a simple project that uses Python Selenium to Login to a Website and scrape some data.
The idea is to provide functions that will make it easier for users to perform certain activities as well as display a simple example using it.

# =====================================
	File Descriptions
# =====================================

scrapper.py -> The main functions file containing all the pre built functionalities
scrapper.yaml -> The config file that is read from for all login details
basic_scrapper.py -> An example of a scrapping program using the above two functions that can be immediately run

# =====================================
	Instructions
# =====================================

 - To install all packages required for this project, clone this project and then run pip install -r requirements.txt
 - To run the example of this project run the basic_scrapper.py file with no changes to any of the files
 - The code is tested on python3
